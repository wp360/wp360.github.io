# github添加、删除远程仓库

git remote用于管理远程仓库

git remote 不带参数时可以参看远程仓库名称

git remote -v 可以查看远程仓库名称和网址

git remote add 仓库名 仓库地址 添加远程仓库，同时设置远程仓库的名字，一般仓库名称是origin，当然你也可以写成其他的名字

git remote rm origin 删除名字为origin的远程仓库

github 创建远程分支以及远程分支无法删除的问题解决
